# Power BI Finance Dashboard

Built an interactive Power BI dashboard using DAX and Power Query to analyze multi-year sales, profit, and growth trends with drill-down, slicers, and KPI visualizations.

---

## Project Overview
- Demonstrates end-to-end Business Intelligence development using Power BI  
- Covers data modeling, advanced DAX calculations, and interactive visualizations  
- Analyzes sales and financial performance across multiple datasets  

---

## Tools & Technologies
- Power BI  
- DAX  
- Power Query  
- Data Modeling  
- Superstore Dataset  
- Finance Dataset  
- Worldwide Box Office Dataset  

---

## Key Objectives
- Compare year-over-year sales using advanced DAX measures  
- Perform ETL and create a unified master dataset  
- Build interactive visualizations and drill-through analysis  
- Implement bookmarks, hierarchies, and conditional formatting  
- Design a fully functional finance dashboard  

---

## Tasks Implemented

### Power BI Platform
- Explored Power BI ecosystem and platform components  

### Year vs Last Year Analysis (DAX)
- Created line and clustered column chart  
- Calculated YoY sales growth using DAX measures  

### Data Modeling & ETL
- Imported multi-year Box Office data  
- Built master table using Power Query transformations  
- Implemented optimized data model  

### Waterfall Analysis
- Developed category-wise yearly waterfall chart using Superstore data  

### Location Hierarchy
- Created Country → State → Region → City → Postal Code hierarchy  
- Visualized location-wise sales performance  

### Bookmarks & Navigation
- Built clustered column chart with bookmark toggles for Year, Quarter, and Month views  

### Drill-Through Analysis
- Designed region-wise profit donut chart  
- Implemented drill-through to detailed West region page  

### Conditional Formatting
- Applied background color scale to Sales  
- Used arrow icons for Profit indicators  
- Applied data bars to Quantity  
- Applied font color rules to Discount  

### Finance Dashboard
- Built interactive dashboard with cards, charts, slicers, and navigation  
- Integrated external website link for enhanced usability
